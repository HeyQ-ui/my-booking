# 中山大学体育场馆预约（演示页）

一个纯前端的中山大学体育场馆预约流程**演示页面**，无需后端、无需安装，双击 HTML 文件即可在浏览器中使用。

> ⚠️ 说明：本项目为静态演示 / 原型，所有数据保存在页面内，**不会真正提交预约**，也不与学校官方系统对接。

## 功能

- 选择预约日期
- 按校区（南校园 / 东校园 / 北校园 / 深圳校区 / 珠海校区）浏览健身房、游泳池场次
- 搜索场馆、选择场次
- 生成「我的预约」结果页，点击预约卡片查看详情悬浮层

## 快速开始

无需任何依赖或构建工具：

1. 下载或克隆本仓库
2. 用浏览器打开 `index.html`
3. 按页面提示依次选择日期 → 场次 → 查看预约结果

也可以用任意静态服务器预览（可选）：

```bash
# Python 3
python -m http.server 8000
# 然后访问 http://localhost:8000/index.html
```

## 使用流程

```
index.html  ──►  booking.html  ──►  我的预约.html
（选日期）        （选场次）          （预约结果 + 详情悬浮层）
```

1. **首页（`index.html`）**：选择预约日期，点击「进入预约页面」。
2. **选场次（`booking.html`）**：确认日期，展开校区折叠栏或用搜索框查找场馆，点击场次卡片，点「确认预约」。
3. **我的预约（`我的预约.html`）**：显示预约结果卡片；点击卡片弹出预约详情悬浮层，点「确认」或点击遮罩关闭。

## 浏览器兼容

面向现代浏览器（Chrome / Edge / Safari / Firefox 近期版本）。使用了 `URLSearchParams`、`Element.closest()` 等较新 API，不支持 IE。
